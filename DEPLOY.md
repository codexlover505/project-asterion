# Публикация ASTERION на GitHub Pages

1. Создайте на GitHub новый пустой репозиторий.
2. Загрузите в корень репозитория всё содержимое этой папки, включая `.github` и `.nojekyll`.
3. Откройте **Settings → Pages**.
4. В разделе **Build and deployment → Source** выберите **GitHub Actions**.
5. Сделайте push в ветку `main` или вручную запустите workflow **Deploy ASTERION to GitHub Pages** на вкладке **Actions**.

После успешной публикации адрес появится в **Settings → Pages** и в завершённом workflow.

## Сохранения

Игра автоматически сохраняется в `localStorage` браузера каждые три секунды, при подборе предметов, победе над врагами, паузе и закрытии страницы. Сохранение привязано к адресу сайта, браузеру и устройству; между разными устройствами оно автоматически не синхронизируется.
